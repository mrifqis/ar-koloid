# AR Efek Tyndall

Folder ini berisi media Augmented Reality berbasis AR.js dan marker Hiro.

Cara pakai:
1. Upload folder ini ke GitHub Pages dengan nama folder `tyndall`.
2. Buka `tyndall/index.html` dari HP.
3. Izinkan akses kamera.
4. Cetak/buka `tyndall/marker.html`.
5. Arahkan kamera ke marker Hiro.
